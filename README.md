# serv-dialogo-redes
Trabalho Final de disciplina de Fundamentos de Redes de Computadores

Requirementos:
- Python 3.10.*
- Docker-compose
- pip 23.3.1

Para Iniciar o projeto:
- Inicie a API: 
    - ``cd serv-dialogo-redes``
    - ``docker-compose up -d``
- Inicie o front-end:
    - ``cd front-end``
    - ``npm install``
    - ``npm start``
